## ğŸš€ Getting Started

### Prerequisites

- A web server (Apache, Nginx, or any local development server)
- PHP support (for contact form functionality)
- Modern web browser

### Installation

1. **Clone or Download** the repository
   ```bash
   git clone [repository-url]
   cd My-Portfolio
   ```

2. **Set up a local web server**
   - Option 1: Use XAMPP, WAMP, or MAMP
   - Option 2: Use Python's built-in server
     ```bash
     python -m http.server 8000
     ```
   - Option 3: Use Node.js http-server
     ```bash
     npx http-server
     ```

3. **Open in browser**
   - Navigate to `http://localhost:8000` (or your server's address)
   - The main page is `index.html`

### Contact Form Setup

The contact form uses PHP to send emails. To make it work:

1. Ensure your server has PHP support
2. The form is located in `forms/contact.php`
3. Configure your email settings in the PHP file if needed
